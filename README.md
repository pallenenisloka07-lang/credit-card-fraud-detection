# Credit Card Fraud Detection

This project detects fraudulent credit card transactions using Machine Learning.

Technologies Used:
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib

Project Description:
This system analyzes credit card transaction data and predicts whether a transaction is fraudulent or genuine.

Steps used in the project:
- Data preprocessing
- Feature engineering
- Handling class imbalance using SMOTE
- Feature scaling
- Training an XGBoost machine learning model
- Evaluating model performance using accuracy, confusion matrix, and ROC-AUC score.
